# Yarn Blocks Mod

A lightweight and aesthetic Minecraft mod that adds **Yarn**, **Yarn Balls**, and **Yarn Blocks** in all 16 wool colors.  
Inspired by real-world knitting and crafting, this mod expands the decorative building options while keeping a vanilla-friendly feel.
